Es un proceso único y es multi-hilo

Soporta ambas tablas transaccionales y no transaccionales

Motores de Almacenamiento

Un motor de almacenamiento es un componente de servidor que ofrece una interfaz entre el cliente y el servidor (a través de sentencias SQL).

Motores de almacenamiento se utilizan para:

Guardar datos

Recuperar datos

Encontrar datos a través de un índice

Procesamiento de dos niveles: Nivel superior incluye analizador de SQL y el optimizador; Bajo nivel se compone de un conjunto de motores de almacenamiento.
